VoiceNavigator Privacy Policy

1. No Data Collection: VoiceNavigator is a "Privacy-First" utility. It does not collect, store, or transmit any personal data, voice recordings, or metadata to the developer or any third-party servers.

2. Local Processing: All speech recognition and command processing occur 100% locally on the user's device. No internet connection is required for the core functionality of the app.

3. Permissions: The app requires Microphone access solely to listen for local navigation commands. This audio stream is processed in real-time and is never recorded or saved.

4. Transparency: As an offline tool, VoiceNavigator operates within the user's local security environment, ensuring total data sovereignty.
