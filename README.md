# 🎙️ VoiceNavigator
### **Offline Voice-Controlled Navigation for macOS**

VoiceNavigator is a privacy-focused utility that lets you control document and application navigation using simple voice commands. Optimized for Apple Silicon (M4/M3/M2/M1).

## 📥 Download
👉 **[Latest Release for macOS & Windows](https://github.com/RahulChandra007/VoiceNavigator-App/releases/latest)**

### 🪟 Windows Installation
1. Download the Windows `.zip` from the link above.
2. Extract the folder and run `VoiceNavigator.exe`.
3. If a "Windows Protected your PC" popup appears, click **More Info** -> **Run Anyway**.

### 🍎 macOS Installation
1. Download the macOS `.zip`.
2. Move to `/Applications`.
3. Run `xattr -cr /Applications/VoiceNavigator.app` in Terminal.

* **100% Offline:** No data ever leaves your Mac. Speech processing happens locally using the Vosk engine.
* **Natural Commands:** Multiple keywords for every action (e.g., "first", "top", "beginning").
* **Ultra-Lightweight:** Designed specifically for macOS architecture to ensure minimal CPU impact.
* **Privacy First:** No cloud APIs, no tracking, and no internet required.
