Reviewer Playbook

Getting Started & App Verification

VoiceNavigator processes speech recognition entirely offline and automates interface navigation natively. Use this script to quickly validate performance.

🔒 No Credentials Required

The application runs entirely locally. It does not require user accounts, registration, cloud server access, or login credentials to function.

🎙️ Prerequisites

Ensure a functional microphone is connected to your PC and configured as the Default System Audio Input Device before launching the tool.

Quick Validation Script

  1. Launch the Engine: Open the executable binary (VoiceNavigator.exe).
  2. Test Single-Instance Guard: With the application running, attempt to open a second copy of VoiceNavigator.exe. Verify that the second instance automatically closes itself to prevent system automation conflicts.
  3. Prepare the Target Interface: Open your document software of choice (such as a PowerPoint deck, Word file, or a multi-page PDF presentation). Make sure your mouse cursor is actively focused on that navigating window.
  4. Issue Voice Commands: Speak clearly into your microphone using these natural localized macros:
    • NEXT: say "next"
    • PREVIOUS: say "previous", "back", "return", or "reverse"
    • RESTART: say "restart", "start", "beginning", "first", or "reset"
    • LAST: say "last" or "finish"
  5. Verify Automation Behavior: Confirm that the active target document or browser view immediately scrolls downward, steps backward, or jumps to anchors in response to your spoken cue.
  6. Pause Listening: Command the assistant to temporarily stop tracking audio feeds. Watch the visual status border around the application frame turn Red to signal listening is disabled.
  7. Resume Listening: Reactivate navigation tracking. Watch the system frame indicator border transition cleanly back to Green.
  8. Graceful Exit: Locate the application icon inside your desktop system tray. Right-click the icon and choose the Quit (X) command to safely shut down background workers and instantly release system resources.
  9. Explore Configurations: Right-click the system tray panel menu anytime to interact with the manual audio level adjustment dashboard, pause/resume mechanisms, and environmental noise auto-calibration modules.
